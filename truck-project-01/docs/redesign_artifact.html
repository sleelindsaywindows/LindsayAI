Backend: two_phase=True flag in session_state.
Phase 1: filter orders where not order.allowed_truck_types (commercial) → solve() → store as phase1_assignments (locked).
Phase 2: remaining orders (builders, allowed_truck_types=["straight"]) → solve() with only straight trucks → phase2_assignments.
Combined display: Phase 1 section grayed+dashed+locked badge, Phase 2 normal cards. Lock prevents UI reorder of Phase 1 stops. Export includes both.
Why two-phase: ensures trailers fill with high-volume commercial stops before builders compete. Homebuilder stops are "straight-only" regardless, so phase split also prevents solver from opening extra trailers to handle builder overflow.
